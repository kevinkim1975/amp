# 제품명
AutoMed Proposal Generator

---

## 1. 개요
병원·의료기관 마케팅 컨설턴트가 JSON 데이터만 작성하면, 30분 이내에 신뢰감 있는 웹 제안서(URL)를 자동으로 생성·배포해 주는 Next.js 기반 서비스입니다. 최종 열람자는 URL을 클릭해 로그인 없이 곧바로 제안서를 확인할 수 있습니다.

---

## 2. 목표 및 KPI
| 구분 | 목표 | 측정 방법 |
|------|------|-----------|
| Primary | 30분 이내 오류 없는 제안서 URL 배포 | 컨설턴트 자체 타이머 + 배포 로그 |
| Secondary | 경영진의 전체 페이지 열람 완료율 >= 90% | Vercel Analytics 페이지 완주율 |

---

## 3. 주요 사용자 & 페르소나
1. 컨설턴트 (System User)
   - 월 2~5회 제안서 생성
   - JSON 작성 -> 빌드 -> 배포 담당
   - 니즈: 시간 절감·자동화·오류 방지
2. 병원 의사결정권자 (End Viewer)
   - URL로 1~3회 열람
   - 니즈: 신뢰감·직관적 데이터·빠른 탐색

---

## 4. Pain Point
1. 반복적 제안서 생성·디자인에 과도한 시간·노동 소모

> 실 작업 1~2일 -> 목표 30분

---

## 5. 주요 사용 사례
1. 신속한 마케팅 제안서 작성·배포 자동화 (Primary)
2. 병원 의사결정권자에게 전문적인 제안서 열람 경험 제공 (Secondary)

---

## 6. 핵심 가치(경영진 시점)
1. 신뢰도 높은 전문 디자인으로 브랜드 이미지 강화
2. 인터랙티브 차트·테이블로 데이터 이해도 향상
3. URL 방식으로 설치 없이 즉시 열람
4. 페이지 네비게이션으로 빠른 정보 탐색
5. 반응형 지원으로 다양한 기기에서 열람 가능

---

## 7. 기능 범위 (MVP)
| 분류 | 기능 | 설명 |
|------|------|------|
| 데이터 검증 | JSON 스키마 검증 | 빌드 시 스키마 불일치 시 실패, 오류 메시지 출력 |
| 페이지 생성 | 동적 페이지(10~20p) & 조건부 렌더링 | 데이터 없는 섹션 자동 제외 |
| 시각화 | 차트·테이블 (Recharts) | 경쟁사 비교, 예산 배분 등 |
| 탐색 | 네비게이션 바 | 이전/다음, 목차, 현재 페이지 3/15 표시 |
| 배포 | GitHub->Vercel 자동 배포 | 커밋 시 자동 빌드·배포, UUID 방식 퍼블릭 URL |
| 디자인 | 신뢰감 있는 UI | Tailwind CSS + shadcn/ui, 반응형 |
| 보안 | 난수(URL) + noindex | 별도 로그인 없이도 무작위 추측 불가 |

### 제외 범위
- JSON 입력 UI
- 템플릿 선택 기능(단일 템플릿 고정)
- 팀 협업(코멘트, 실시간 편집 등)
- PDF 내보내기
- 영상/이미지 대용량 업로드
- AI 요약/카피 생성
- 결제 연동

---

## 8. 기술 스택
- Next.js 15 (App Router)
- React 18 + TypeScript
- Tailwind CSS + shadcn/ui
- Recharts (차트)
- 배포: Vercel, GitHub Actions(옵션)

---

## 9. 개발 전략
MVP -> 점진적 기능 추가(Agile)
1. Phase 1: MVP 전체 기능 구현·내부 테스트
2. Phase 2: 실제 제안서 1건으로 검증, 피드백 반영
3. Phase 3: 사용 데이터 기반 점진적 개선 및 부가 기능 추가

## 10. 비기능 요구사항
| 항목 | 목표 |
|------|------|
